# 匂い研究 VAS アプリ

VAS を独立運用するための iPad 向け静的 Web アプリです。10cm 校正、5 本の VAS 入力、端末内保存、Excel 互換 `.xls` 出力に対応しています。

## できること

- 参加者情報を入力して VAS セッションを開始
- 最初に 10cm 校正を実施
- 5 項目の VAS を 1 本ずつ入力
- 患者画面では数値を表示せずに位置だけ選択
- 履歴を端末内に保存
- 単回または履歴まとめて Excel 互換 `.xls` 出力

## 使い方

1. `index.html` を iPad の Safari で開く
2. 参加者 ID を入力して開始
3. 校正画面で青い線が実寸 10cm になるよう一度だけ調整
4. 各 VAS で線上をタップまたはドラッグ
5. 最後の確認画面から保存または Excel 出力

## 保存について

- 履歴、下書き、校正値は `NioiQuestionnaire` とは別キーで `localStorage` に保存されます
- 校正値はこのブラウザに保存され、次回以降も引き継がれます

## 出力内容

- 概要シート: 参加者情報、保存日時、校正値、各 VAS の mm 値
- 生データシート: 各設問の左右ラベルと raw 値
